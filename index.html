Calmati cafetta

制作期間
企画:2日
デザイン:1週間
コーディング:3週間
目的
カフェの認知拡大
ターゲット
友人とランチが好きな20~30代の女性
使用言語/ソフト
HTML,CSS,javascript
(jQuery)photoshop,
Adobe XD,visual studio code
コメント
落ち着いたカフェをイメージして茶色系の色をメインに温かみのあるサイトにしました。予約フォームでは離脱率を防ぐために最低限の情報の入力だけで済ませられるように工夫しました。
改善点
  • ヘッダーの背景に薄い線を入れて文字を見やすくしました。
  • フォントカラーが真っ黒だと目立ってしまうため、メインカラーを取り入れました。
